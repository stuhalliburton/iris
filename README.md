# Iris Classification

## Dataset

[UCI Machine Learning: Iris
Dataset](https://archive.ics.uci.edu/ml/datasets/Iris)

## Setup

```bash
pip install -r requirements.txt
```

## Running

To classify Iris' via k nearest neighbors

```bash
python knn.py
```


To classify Iris' via neural network

```bash
python train_nn.py
```
